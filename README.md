# vue-manage-system
基于Vue.js 2.x系列 + element-ui 内容管理系统解决方案

简单用法
（1）安装依赖 npm install
（2）运行 npm run dev
（3）在浏览器输入 http://localhost:8080/#/
（4）浏览器如果正常输出，运行成功
